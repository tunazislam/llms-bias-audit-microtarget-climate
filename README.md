# llms-microtarget-climate
This repository contains code and data for the paper titled "[Who Gets Which Message? Auditing Demographic Bias in LLM-Generated Targeted Text
](https://arxiv.org/pdf/2601.17172)"

## Necessary Libraries
The code is implemented in Python 3. 

Other major libraries needed:

Python 3.9.19

PyTorch 1.13.1

jupiter notebook

pandas

gensim

nltk

nltk.tag

spacy

emoji

sklearn

scipy

matplotlib

numpy

preprocessor

transformers

seaborn

BERTopic

tiktoken

openai

groq

mistralai

## Directory Structure:
- `Data/`: Contains all datasets.
- `Code/`: Contains scripts for analysis.
